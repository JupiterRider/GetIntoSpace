# GetIntoSpace

## Controls:
- <b>A/D</b> or <b>Left/Right</b> to move sideways
- <b>W</b>, <b>Space</b> or <b>Up</b> to accelerate
- <b>ESC</b> to quit

## Gameplay:
The aim is to survive as long as possible without a collision. As time goes on, the asteroid gets faster and faster.
